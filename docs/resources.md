# Resources

This page provides key resources for both students and TAs.  
Always check the official Clinic website first for the most up-to-date information: [Data Science Clinic Website](https://clinic.ds.uchicago.edu)

## For Students
- **Official Clinic Website:** [https://clinic.ds.uchicago.edu](https://clinic.ds.uchicago.edu)  
  Primary hub for course information, deliverables, and policies.  
- **Canvas:** Weekly reports, attendance tracking, and grading.  
- **Technical Help:**  
  - GitHub documentation and tutorials.  
  - Docker and Make documentation.  
  - Python style and best practices (PEP 8).  
- **Campus Services:**  
  - [Undergraduate Academic Support](https://college.uchicago.edu/academics/college-core-tutor-program) - College Core Tutor Program for STEM courses  
  - [Graduate Academic Support](https://grad.uchicago.edu/academic-support/) - UChicagoGRAD resources and workshops  
  - [Student Wellness](https://wellness.uchicago.edu/) - Counseling and wellness support  

## For TAs
- **Clinic Website:** [https://clinic.ds.uchicago.edu](https://clinic.ds.uchicago.edu)  
  For official coding standards, policies, and grading procedures.  
- **TA Training Docs (this repo):** Teaching basics, professionalism, and session guides.  
- **Checklists:**  
  - Attendance reporting (Canvas).  
  - [Repo evaluation checklist](https://clinic.ds.uchicago.edu/rubrics/final-technical-cleanup.html) - Final technical cleanup rubric.  
- **Mentorship Support:**  
  - Point of contact: assigned project mentor.  
  - Escalation: Clinic Director.  

## Recommended References
- **Python Development:**  
  - PEP 8 – Style Guide for Python Code.  
  - “Effective Python” by Brett Slatkin.  
- **Version Control:**  
  - [GitHub Flow Guide](https://docs.github.com/en/get-started/quickstart/github-flow) - Branching and pull request workflow  
  - [GitHub Pull Request Best Practices](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/getting-started/best-practices-for-pull-requests) - Code review practices  
  - [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials) - Comprehensive Git learning resource  
- **Reproducibility Tools:**  
  - [Docker Documentation](https://docs.docker.com/) - Containerization and reproducible environments  
  - [Docker Best Practices](https://docs.docker.com/develop/dev-best-practices/) - Writing efficient Dockerfiles  
  - [uv Documentation](https://docs.astral.sh/uv/) - Fast Python package manager and resolver  
  - [GNU Make Manual](https://www.gnu.org/software/make/manual/make.html) - Build automation and task running

## Notes
- Students should **consult the Clinic website before asking TAs** for answers.  
- TAs should **refer back to official documentation** for policies (attendance, grading, repo standards).
